Validates the data bytes using CRC checksum algorithm.

After copying the given data, CRC checksum is calculated for the given polynomial. If the evaluated result is equal to the given value then the validation is successful.

Parameters
dataPointer to data bytes to be validated
dataSizeSize of the data
polynomialWithoutMsbCRC polynomial with the most significant bit discarded
crcValueValue to be checked with the calculated checksum
Returns
